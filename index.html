About Coding Competition

Coding Competition test participants' skills and reasoning in solving computer programs to address the given problems. The evaluation criteria include the speed of writing the program and the accuracy/efficiency of the program created for each related problem case.

During the competition, participants will be given descriptions of several problems, and within the allotted time, they must write and submit as many programs as possible that can solve each of those problems.

Each program created must not only correctly solve the given problem case but also run within a very limited time for each case. Therefore, besides competing in programming speed, participants are also required to find and use the appropriate and efficient algorithms (and data structures). The programming languages used are Java, C++, and Python.
